======
Issues
======

Thank you for helping improve the project by opening an issue. Please consider the following guidelines to enable efficient troubleshooting:

1. Provide detailed steps to reproduce the problem
2. Include system information (OS, RAM, GPU)
3. Attach log files if available
4. Share a minimal example file that demonstrates the issue
5. Note Mosaic version and relevant settings

Issues can be submitted on `GitHub <https://github.com/KosinskiLab/pytme/issues>`_.

.. tip::

	Feel free to open an issue if you are missing a feature.
