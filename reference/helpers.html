class Ntree(arr)[source]#

Bases: object

N-dimensional dyadic tree.

Each array dimension is split into two similarly sized halves. The amount of subvolumes per split equals 2**n with n being the dimension of the input array.

Attributes:
nleavesint

Number of leaves in the Ntree.
